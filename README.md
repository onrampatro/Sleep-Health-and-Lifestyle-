# Sleep Health and Lifestyle Data Analysis

This project focuses on analyzing the relationship between sleep health and lifestyle factors using data science techniques and data visualization tools.

## Objective

The objective of this project is to examine how lifestyle behaviors, including physical activity, stress levels, and occupation, influence sleep quality and overall health.

## Tools and Technologies

- Python (Pandas, Matplotlib, Seaborn)
- Google Colab
- Looker Studio (for dashboard visualization)

## Dataset

- Source: Kaggle
- Dataset: Sleep Health and Lifestyle Dataset
